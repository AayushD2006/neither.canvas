# Neither

**Neither** is a personal infinite canvas note-taking application inspired by Notion, Obsidian Canvas, and Excalidraw. It is designed to help organize ideas visually using draggable note bubbles, connections, and nested canvases in a distraction-free environment.

## ✨ Features

* Infinite zoomable and pannable canvas
* Draggable and resizable note bubbles
* Connect multiple bubbles with customizable arrows
* Nested canvases (Canvas inside Canvas)
* Rich text, code snippets, images, and to-do lists
* Dark theme with customizable backgrounds
* Responsive layout for different screen sizes
* Automatic local saving using browser storage
* Export and import project backups

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)

## 🚀 Getting Started

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Start creating and organizing your ideas.

No installation or additional setup is required.
